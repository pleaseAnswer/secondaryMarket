### 创建项目 vue create '项目名'

##### 安装依赖

* element-ui  
  
> cnpm i element-ui -S

* vue-router  

> cnpm i vue-router -S

* axios  

> cnpm i axios -S

* vuex  

> cnpm i vuex -S

* sass  

> cnpm i node-sass -D
> cnpm i sacc-loader -D
